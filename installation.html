2. Instalation

  1. Mod
    • Follow the instructions on how to install your prefered modloader (be it Forge, Fabric, NeoForge or Quilt)
    • Drag the downloaded .jar file into (on windows) %appdata%\.minecraft\mods
    • Launch the world and start playing

  2. Datapack
    • Locate the datapack folder for the specific world
    • Drag the Magic_entire.zip file into the datapack folder
    • Drag the Magic_resourcepack.zip file into the resourcepack folder
    • Enable the datapack with /datapack enable "file/Magic_entire.zip"
    • Relog the world

    • Alternatively you can apply the datapack when creating the world
    • Then you neither have to enable the datapack nor exit and re-enter the world
