2. Installation

  • I downloaded the datapack version

    1. Locate your datapack folder

    If your world is already existing this folder can be found by selecting the world in the menu, then pressing "Edit", followed by "Open world folder", then open the "datapacks" folder.

    If you want to create a new world go to the "More" tab, then press "Data Packs" and then "Open Pack Folder".

    2. Move the datapack

    Copy the Magic.zip file you downloaded into the datapack folder.

  • I downloaded the mod for Forge / Fabric / NeoForge / Quilt

    1. Download the Mod launcher of your choice

    Be it the Forge, Fabric, NeoForge or Quilt -launcher.

    2. Locate your mods folder

    The mod folder is located within the Minecraft appdata folder.

    It can be found on Windows by typing WINDOWS+R, then %appdata%\.minecraft\mods.

    Alternatively, you can go to the world menu, press "Edit" on a world, followed by "Open world folder" and then go up one folder and then down into "Mods".

    3. Move the mod

    Copy the Magic.jar file you downloaded into the mods folder.
